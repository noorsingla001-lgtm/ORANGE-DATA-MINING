# 🍊 Orange Data Mining Workflow

An open-source visual data science and machine learning pipeline built using **Orange Data Mining**. This project showcases data analysis, feature selection, predictive modeling, and model evaluation through an intuitive, drag-and-drop workflow interface without requiring extensive coding.

---

## 🌟 Key Features

* **Visual Pipeline Integration:** Completely code-free data science workflow utilizing standard Orange widgets.
* **Feature Selection:** Filter and select specific variables to streamline predictive model training.
* **Predictive Modeling:** Implements a **Logistic Regression Model** for binary or multi-class classification tasks.
* **Performance Evaluation:** Built-in validation using testing and scoring mechanisms to assess model accuracy and metrics (such as AUC, Precision, and Recall).
